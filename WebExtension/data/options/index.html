1: If this option is checked and [2] is unchecked, the YouTube player is released from the no-buffer state only once.
2: If this option is checked and [1] is checked too, the YouTube player gets paused when the tab is hidden and it gets resumed when the tab is active again.
3: Recently, Ad-blocker extensions do not allow YouTube from fetching video information when the page is already loaded, so stopping the video is not working anymore since the player needs to fetch the video information later. To be compatible with ad-blocker extensions, this extension by default pauses videos instead. If you have whitelisted the blocking rule on your ad-blocker extension, uncheck this option for real no-buffering.
